API Documentation

Description: Generate a regular temporary email address.

Use Case: Create a disposable email for general use cases.

Parameters: None

Example Request:

GET /api/gen

Example Response:

{
  "api_owner": "@ISmartCoder",
  "api_dev": "@WeSmartDevelopers",
  "temp_mail": "example@temp-mail.org",
  "access_token": "your-access-token",
  "time_taken": "0.12s",
  "expires_at": "N/A"
}

Response Fields:

  • api_owner: API creator handle.
  • api_dev: Developer team handle.
  • temp_mail: Generated email address.
  • access_token: Token to check messages.
  • time_taken: Time to generate email.
  • expires_at: Expiration time (N/A for regular emails).

Description: Check messages for a regular temporary email using the provided token.

Use Case: Retrieve emails received by the generated temporary email.

Parameters:

Parameter Type Description
token String Access token from /api/gen response.

Example Request:

GET /api/chk?token=your-access-token

Example Response:

{
  "mailbox": "example@temp-mail.org",
  "messages": [
    {
      "from": "sender@example.com",
      "subject": "Test Email",
      "body": "This is a test email.",
      "receivedAt": "2025-09-13 21:03:00",
      "api_dev": "@ISmartCoder",
      "api_updates": "@WeSmartDevelopers"
    }
  ],
  "api_owner": "@ISmartCoder",
  "api_dev": "@WeSmartDevelopers"
}

Response Fields:

  • mailbox: Email address.
  • messages: Array of received messages.
  • api_owner: API creator handle.
  • api_dev: Developer team handle.

Description: Check messages for an .edu email using the provided token.

Use Case: Retrieve emails received by the .edu email.

Parameters:

Parameter Type Description
token String Access token from /api/edu/gen response.

Example Request:

GET /api/edu/chk?token=your-access-token

Example Response:

{
  "api_owner": "@ISmartCoder",
  "api_dev": "@TheSmartDev",
  "edu_mail": "example@edu.etempmail.com",
  "access_token": "your-access-token",
  "messages": [
    {
      "From": "sender@example.com",
      "Subject": "Welcome to Edu Mail",
      "Date": "2025-09-13",
      "body": "This is the email body.",
      "Message": "This is the email body."
    }
  ],
  "Message": "This is the email body.",
  "From": "sender@example.com",
  "body": "This is the email body.",
  "Date": "2025-09-13",
  "Subject": "Welcome to Edu Mail"
}

Response Fields:

  • api_owner: API creator handle.
  • api_dev: Developer team handle.
  • edu_mail: Email address.
  • access_token: Token used for the request.
  • messages: Array of received messages.
  • Message, From, body, Date, Subject: Details of the latest message (if any).
